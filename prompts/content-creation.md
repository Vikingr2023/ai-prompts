# 📝 内容创作提示词模板

## 📌 公众号文章模板

```
帮我写一篇公众号文章：

标题：[标题方向，越具体越好]
主题：[你要写什么]
读者：[谁会读这篇文章]
风格：[干货/故事/热点/教程]

要求：
- 开头用故事或痛点引入（300字内）
- 中间3-5个要点，每个要点有案例支撑
- 语言口语化，像在和人聊天
- 结尾有明确的行动号召
- 加入1-2个金句
- 全文1500-2500字
```

## 📌 视频号脚本模板

```
帮我写一个视频号脚本：

时长：[X]分钟
主题：[主题]
目标：[看完后观众要做什么]

结构：
- 前3秒：Hook，如何让人停下来
- 中间：[核心内容，2-3个要点]
- 最后：CTA，关注/评论/链接

要求：
- 口语化，像在说话不是朗读
- 每段不超过3行
- 留一个互动点（问句或征集）
```

## 📌 朋友圈文案模板

```
帮我写[数量]条朋友圈文案：

主题：[主题]
风格：[多选：幽默/专业/温暖/焦虑/成就展示/日常分享]
用途：[销售/种草/互动/品牌]

每条要求：
- 不超过200字
- 有[emoji]
- 有真实感，不要太营销
- 最后带CTA

格式：
1. [文案1]
---
2. [文案2]
---
```

## 📌 短视频口播模板

```
帮我写一段口播文案：

时长：1-3分钟
主题：[主题]
场景：[知识科普/好物推荐/经验分享/热点点评]

要求：
- 第一句必须是让人想看下去的
- 中间有转折或意外
- 结尾有金句或互动
- 全程用"我"的口吻，像在和人聊天
```

## 📌 案例故事模板

```
帮我写一个案例故事：

主角：[客户背景，公司/职业/痛点]
用了什么：[产品/方法/工具]
效果：[具体数字，越具体越好]

结构：
1. 痛点引入（他遇到了什么问题）
2. 转折点（怎么接触到这个解决方案的）
3. 过程（具体做了什么）
4. 结果（拿到什么成果）
5. 感悟（他怎么说）

要求：
- 真实感，不要太完美
- 有细节，有情绪
- 1000-2000字
```
