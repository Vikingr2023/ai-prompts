# 💼 商业布道话术提示词

## 📌 布道开场白模板

```
你是算力老何，一个有24年创业经验的AI布道师。
帮我写一段面向[目标人群]的AI布道话术，
主题是[主题]，时长[时长]分钟。

要求：
- 面向小白、零术语、接地气
- 开头要有hook，3秒抓住注意力
- 中间有2-3个真实案例
- 结尾有明确的行动号召
- 全程口语化，像在和人聊天
```

## 📌 布道核心话术结构

### 1. 开场Hook（30秒）

```
我叫何文强，创业24年。
2024年，我用AI赚了第一个100万。
今天我不讲概念，只讲我实际怎么做。
```

### 2. 痛点共鸣（1分钟）

```
你是不是也有这些困扰？
- 每天忙得脚不沾地，但好像没赚到什么钱
- 想用AI，但不知道从哪儿开始
- 试过一些工具，但感觉没什么用
如果有同感，评论区扣1
```

### 3. 解决方案展示（3-5分钟）

```
我有个客户，原来每天花3小时处理客服问题。
用了我教他的AI方法，现在只需要30分钟。
不是他更努力了，是他的AI助手帮他干了大部分活。
```

### 4. 行动号召（30秒）

```
如果你也想试试，
可以扫码用一下这个工具，
第一次用是免费的。
觉得有用再继续，不满意随时走。
```

## 🎯 不同场景的话术模板

### 场景1：一对一销售转化

```
帮我写一段一对一销售话术，
客户类型：[客户类型]
产品：[产品名称]
客户顾虑：[顾虑1]、[顾虑2]、[顾虑3]
要求：先共情，再解决问题，最后自然推出产品
```

### 场景2：线下培训开场

```
帮我写一个[时长]分钟的培训开场白，
学员是[学员群体]，
主题是[主题]，
要求：有趣、有料、有互动设计
```

### 场景3：朋友圈文案

```
帮我写一条朋友圈文案，
主题是[主题]，
风格：[幽默/专业/温暖/焦虑]
长度：不超过200字
要有[emoji]点缀
最后加一个CTA
```
